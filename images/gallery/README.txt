■ 「過去の活動」ページのフォトギャラリーの変え方

このフォルダ(images/gallery)に写真を入れると、
「過去の活動」ページの活動フォトギャラリーに表示されます。

1. 写真(.jpg / .png / .webp)をこのフォルダにコピーする
2. 一番上のフォルダにある「サイトを更新.bat」をダブルクリックする
3. 1〜2分後にサイトに反映される

・フォルダ内の写真がすべて、ファイル名順に並びます。
  「01-能登.jpg」「02-大槌町.jpg」のように番号を付けると順番を指定できます。
・枚数の制限はありません。写真を減らしたいときはフォルダから削除してください。
・各写真は横4:縦3の枠に自動トリミングされて並びます。
