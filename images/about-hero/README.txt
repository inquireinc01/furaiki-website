■ 「過去の活動」ページのメインビジュアルの変え方

このフォルダ(images/about-hero)に写真を入れると、
「過去の活動」ページ上部のメインビジュアル(背景写真)になります。

1. 写真(.jpg / .png / .webp)をこのフォルダにコピーする
2. 一番上のフォルダにある「サイトを更新.bat」をダブルクリックする
3. 1〜2分後にサイトに反映される

・使われるのは「ファイル名順で先頭の1枚」です。
  差し替えたいときは、古い写真を削除して新しい写真を入れるか、
  「01-○○.jpg」のように先頭に来る名前を付けてください。
・横長の写真がきれいに表示されます。
・iPhoneの写真(HEIC形式)もそのまま入れてOKです。
  「サイトを更新.bat」実行時に自動でJPGに変換されます。
