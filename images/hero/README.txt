■ メインビジュアル写真の入れ方

このフォルダ(images/hero)に写真を入れると、
トップページの横に流れるメインビジュアルに反映されます。

1. 写真(.jpg / .png / .webp)をこのフォルダにコピーする
2. 一つ上のフォルダにある「サイトを更新.bat」をダブルクリックする
3. 1〜2分後に https://www.furaiki.org に反映される

・表示順はファイル名の順番です。
  「01-能登.jpg」「02-大槌町.jpg」のように番号を付けると順番を指定できます。
・横長の写真がきれいに表示されます(縦横は自動でトリミングされます)。
・写真を減らしたいときは、このフォルダから削除して同じく bat を実行してください。
・iPhoneの写真(HEIC形式)もそのまま入れてOKです。
  「サイトを更新.bat」実行時に自動でJPGに変換されます。
